Pizza Sales Analysis using SQL

--
- Designed and queried a relational database containing orders, pizzas, categories, and pricing data.
- Wrote SQL queries using SELECT, WHERE, GROUP BY, ORDER BY, and HAVING clauses.
- Used INNER JOIN operations to combine multiple tables for analysis-ready datasets.
- Calculated business KPIs such as total revenue, total orders, and quantity sold.
- Performed category-wise, size-wise, and product-wise sales analysis.
- Identified top and bottom performing pizzas based on revenue and sales volume.
- Analyzed peak sales hours, weekday trends, and monthly order patterns.
- Applied aggregate functions, subqueries, and window functions for advanced insights.
- Created SQL Views for cleaned, structured, and reusable reporting datasets.
- Integrated SQL Views with Microsoft Power BI for dashboard development and visualization.
